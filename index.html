This hub now runs on a real shared database (Firebase Firestore) so everyone visiting sees the same live data. To finish setup:

  1. Go to console.firebase.google.com and create a free project.
  2. Inside it: Build → Firestore Database → Create database (test mode is fine to start).
  3. Project settings (gear icon) → scroll to "Your apps" → add a Web app → copy the firebaseConfig object it shows you.
  4. Open this file's code and paste those values into the FIREBASE_CONFIG object near the top of the <script> section.

There's a suggested Firestore security rule in a comment right above that config block too — worth reading before you go live.
